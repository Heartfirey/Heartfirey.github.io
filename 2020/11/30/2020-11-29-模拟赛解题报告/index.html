1.锯齿文本

题目分析：题目要求按锯齿形状逐行递增递减的输出字符，那么就涉及两个状态量：递增控制、递减控制，由于递增的上限是固定的，我们只需设置一个循环，循环的自增量用变量表示，对于递增的过程，每轮循环结束后的增量为1，递减则为-1。那么问题就被简化成了独立输出每一行字符的问题。

Python代码

a = int(input())
n = int(input())
sj = "JXNU "
def printjxnu():
    cnt = 0
    flag = 1
    for i in range(0, n):
        if(cnt == 1): flag = 1
        if(cnt == a): flag = -1
        cnt += flag
        for j in range(0, cnt): print(sj, end = "")
        print()
printjxnu()
exit()

2.HelloWorld!

题目分析：显然，给出的字符串都固定了，只需要转换大小写即可。至于插入AI，在循环输出的时候判断一下索引的位置即可。

Python代码：

a = str(input())
for i in range(0, len(a)):
    if(i == 0): print(a[i].upper(), end = "")
    elif(i == 5): print(a[i].upper(), end = "")
    else: print(a[i].lower(), end = "")
    if(i == 4): print("AI", end = "")

C/C++:(C自行修改)：

#include <iostream>
using namespace std;

int main()
{
    string str;
    cin >> str;
    for(int i = 0; i < str.size(); i++){
        if(i == 0) str[i] = toupper(str[i]);
        else if(i == 5)  str[i] = toupper(str[i]);
        else str[i] = tolower(str[i]);
        cout << str[i];
        if(i == 4) cout << "AI";
    }
}

3.求a * aa * aaa…

题目分析：模拟产生2、22、222的过程即可

Python：

n = int(input())
sum = 0
k = 1
for i in range(0, n):
    sum = sum * 10 + 2
    k *= sum
print(k)

4.沙漏

N=int(input())
c=input()#读入字符

if N<=6: #N<=6时直接特判输出
    print(c)
    print(N-1)
else:
    h=int(((N+1)/2)**0.5)#这里自己推一遍行数h和N的关系，慢慢推别慌，就数学规律
    
    #打印第一个倒三角形
    for i in range(0,h):
        for j in range(0,i):
            print(" ",end="") #输出空格
        for k in range(0,2*(h-i)-1):
            print(c,end="")
        print()
        
    #打印第二个正的三角形
    for i in range(1,h):
        for j in range(0,h-i-1):
            print(" ",end="")
        for k in range(0,2*i+1):
            print(c,end="")
        print()
        
    #输出剩下的
    print(N-2*h*h+1)
