2.Fibonacci数列

本题目主要练习“递推”方法的使用，设置a，b为前两项，使用中间变量临时存储b的值，再分别将a,b向下递推，求出结果即可

M = 10007
n = int(input())
a = b = 1
for i in range(1, n):
    temp = b
    b = int((a + b) % M)
    a = temp;
print(a)

3.复数乘积

Python自带复数数据类型，期中考试有提到可以用a + bj或a + bJ的形式来表示复数，但这要求a，b为常量，这里显然要用变量，因此就要使用complex方法将实虚部变量转换为复数表达式，再进行运算

a, b = map(int,input().split())
c, d = map(int,input().split())
e = complex(a, b)
f = complex(c, d)
e *= f
print(int(e.real), int(e.imag))

4.复数加法

跟上题几乎一样，就是换个运算方法

a, b = map(int,input().split())
c, d = map(int,input().split())
e = complex(a, b)
f = complex(c, d)
e += f
print(int(e.real), int(e.imag))

P1 求最大公约数

注意：python默认允许递归深度为1000层，超出1000层会判定为堆栈溢出，因此使用方法一时需要手动修改最大允许的递归深度（法二不需要）

import sys   
sys.setrecursionlimit(100000)

代码：

#import sys   
#sys.setrecursionlimit(100000)
#方法一
def gcd_one(a,b):
    if b > a:
        a, b = b, a
    if a == b:
        return a
    elif a - b > b:
        return gcd_one(a - b,b)
    else:
        return gcd_one(b,a - b)
#方法二
def gcd_two(a,b):
    if b > a:
        a,b = b,a
    if a % b == 0:
        return b
    else:
        return gcd_two(b,a % b)

a, b = map(int, input().split())
print(gcd_one(a, b))
