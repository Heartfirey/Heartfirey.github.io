4.呵，某多校招题

本题选自2020年江西师范大学18级新风杯 3

对于$1\ 2\ 3\ 4\ 5\ 6\ 7\ 8$这一组数，

第一次我们选择$x = 4$得到$1\ 2\ 3\ 4\ 1\ 2\ 3\ 4$，

第二次我们选择$x = 2$得到$1\ 2\ 1\ 2\ 1\ 2\ 1\ 2$,

第三次我们选择$x = 1$得到$1\ 1\ 1\ 1\ 1\ 1\ 1\ 1$,

第四次我们选择$x = 1$得到$0\ 0\ 0\ 0\ 0\ 0\ 0\ 0$。

总共四次操作，我们观察8的二进制表示$1000$，共有$4$位，并且当$n \in [1000_2, 10000_2)$之间时，操作次数也是4次，所以就有规律了。

t = int(input())
for j in range(t):
    n = int(input())
    ans = 0
    for i in range(60, -1, -1):
        if(n >> i & 1):
            ans = i + 1
            break;
    print(ans)

5.数字黑洞

本题思路很简单，实现比较复杂，但只要仔细阅读题目，按题意进行运算并计数，写一下基本没问题

s=input().split()[0]
n=0
def loop(s):
    global n
    if int(s)==6174:
        print(n)
    else:
        L=[int(i) for i in list(s)]
        L.sort()
        re1=re2=0
        for i in range(4):
            re1+=L[i]*10**(i)
            re2+=L[::-1][i]*10**(i)
        re=re1-re2
        n+=1
        if len(list(str(re)))==4:
            loop(str(re))
        else:
            while len(list(str(re)))<4:
                re='0'+str(re)
            loop(re)
loop(s)

6.交换Easy

首先采用a,b=N,M
我们不需要限制a的数字，可以直接通过列表来保存输入的内容，然后循环b次，每次获取2个索引值，然后交换他们对应的元素，需要注意的就是索引是从0开始，所以需要-1操作。最后循环输出即可。

a, b = map(int, input().split())
indexs = list(map(int, input().split()))
for i in range(b):	#循环b次
    p, q = map(int, input().split())	#每次获取索引
    indexs[q-1], indexs[p-1] = indexs[p-1], indexs[q-1]
for i in indexs: print(i)

P1

遍历查找，注意只需要查找到$\sqrt{n}$即可，可由均值不等式证明合法的解一定$\le\sqrt{n}$

import math
def sq_number(n):
    if n < 0:
        return False
    for i in range(int(math.sqrt(n)),-1,-1):
        if i * i == n:
            return True
        j = n - i * i
        k = int(math.sqrt(j))
        if k * k == j:
            return True
        return False
n = int(input())
print(sq_number(n))
